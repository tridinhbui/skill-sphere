# SkillSphere

A Global Solutions Senior Capstone Project dedicated to empowering career success through free, accessible resources for students and graduates worldwide.

## Project Overview

SkillSphere addresses the critical challenges facing today's students and graduates as they enter an increasingly competitive job market dominated by AI. Our platform provides comprehensive, free resources to help individuals at all education levels strengthen their resumes, find internships, and increase their career opportunities.

## The Challenge

- **Lack of Job Opportunities**: Students and graduates face limited entry-level positions
- **AI Domination**: Entry-level jobs are increasingly being replaced by AI
- **Unequal Distribution**: Gender disparities persist across various career paths
- **Lack of Real-World Guidance**: Employment needs aren't discussed enough in education

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Build for Production

```bash
# Create an optimized production build
npm run build

# Start the production server
npm start
```

## Technology Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel (recommended)
